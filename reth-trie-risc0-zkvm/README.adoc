= Reth Trie Risc0 zkVM

Ethereum state, consisting of tries based on Risc0's MPT, but constructable from Reth execution witness.

== Compatibility with `no_std`

Once a while run the following to make sure this crate is **truely** `no_std` compatible:

[source,shell]
----
cargo build -p reth-trie-risc0-zkvm --target thumbv7em-none-eabi
---
