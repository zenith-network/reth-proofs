= Reth Proofs - Ethereum Prover with Ethproofs Integration

== Usage

== Proving Single Block

. Prepare block input:
+
[source,shell]
----
cargo run -p reth-proofs-gevulot \
  prepare-block \
  --block-number 22789000 \
  --http-rpc-url http://130.250.187.55:8545
----
+
NOTE: You can generate input up to 100 blocks behind the chain tip, because getting witness of old blocks is getting too slow in Reth node.
+
At this point you will get file with block input placed at `/tmp/reth_proofs_cache/input/`.

. Prove block:
+
[source,shell]
----
cargo run -p reth-proofs-gevulot \
  prove-block-offline \
  --block-input /tmp/reth_proofs_cache/input/22789000.bin \
  --proving-key-path ./pk.bin
----

== Core Compatibility with `no_std`

Once a while run the following to make sure Reth Proofs core is **truely** `no_std` compatible:

[source,shell]
----
cargo build -p reth-proofs-core --target thumbv7em-none-eabi
---
